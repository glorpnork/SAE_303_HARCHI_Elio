Vous trouverez mon site qui contient 6 graphiques interactifs.
Pour lancer le site, il suffit d'ouvrir le fichier index.html dans un navigateur web.
lorsque vous ouvrez le site, vous verrez 6 graphiques.
vous pouvez interagir avec les graphiques en survolant les points ou les barres.
vous pouvez aussi zoomer et dézoomer sur les graphiques.
vous pouvez aussi sauvegarder les graphiques en cliquant sur le bouton de sauvegarde.
et lorsque vous appuyez sur le bouton "analyse" vous verrez les analyses des graphiques.
